# IRON · 45

Programme d'entraînement adaptatif sans matériel. 8 semaines, 45 minutes par jour.

## Stack

- Vite + React 18 + TypeScript
- React Router
- CSS natif avec variables pour theming dual (dark / light)

## Développement

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Déploiement

Déployé automatiquement sur Vercel à chaque push sur `main`.
